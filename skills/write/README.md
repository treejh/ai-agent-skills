# write

중국어 또는 영어 산문을 작성하거나 편집하거나 다듬을 때 사용하는 스킬. AI 글쓰기 패턴을 제거하고 자연스러운 문체로 다시 작성한다.

> 출처: [obra/superpowers](https://github.com/obra/superpowers) (v3.8.0)

## 기능

- 텍스트 언어 자동 감지 (한자 포함 → 중국어 규칙, 그 외 → 영어 규칙)
- AI 특유의 어투 패턴 제거
- 의미를 바꾸지 않고 자연스러운 문체로 재작성
- 구조 변경 없이 내용 in-place 편집
- 독자층이 불분명하면 먼저 확인 (초급 엔지니어 vs 시니어 아키텍처)

## Hard Rules

- **의미 우선, 스타일 후순위** — AI 패턴 제거가 의미를 바꾸면 원문 유지
- **명시적 요청 없이 삭제 금지** — 불필요해 보여도 플래그만 표시, 삭제 안 함
- **조용한 구조 변경 금지** — 헤딩 재정렬, 단락 순서 변경은 요청 시에만
- 출력 후 설명 없음 — 사용자가 물어볼 때만 변경 이유 설명

## 설치

```bash
git clone https://github.com/treejh/ai-agent-skills.git ~/ai-agent-skills
ln -s ~/ai-agent-skills/skills/write ~/.claude/skills/write
```

## 사용법

```
/write
```

편집할 텍스트를 붙여넣거나 파일 경로를 지정한다.

## 언제 사용하나

- 기술 블로그 포스트 다듬기
- PR 설명이나 커밋 메시지 산문 품질 개선
- 슬랙/이메일 메시지 자연스럽게 수정
- `/learn` Phase 5에서 자동으로 호출됨

> **사용 안 할 때**: 코드 주석, 커밋 메시지, 인라인 문서 작성 (코드 관련 텍스트에는 사용하지 않음)
